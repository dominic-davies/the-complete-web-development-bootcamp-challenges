Adding behaviour to websites

We started with some simple alerts. The full syntax is window.alert("hello"); alert is a special keyword (function) gets the browser to do something. We use JavaScript's syntax/grammar so the computer can understand what we are telling it to do.

alert('Hello');

alert('World!');

Data Types

The "quotes" tell JS that it's a string. A string of characters (text).

typeof("Dominic"); JS tells you it's a string data type.

Variables

we get the browser to ask prompt('What is your name?'); if we type our name in the alert box - it doesn't get stored and we lose it.

We can alert myName -

var myName = 'Dominic';

alert(myName);

So we can ask for your name and save it as a variable.

var yourName = prompt("What is your name?");

alert("Your name is " + yourName);

Variable naming (conventions)

You can name your var anything but it should make sense and not be a reserved keyword. You will work with other developers so give meaningful names to your variables.

Some good practices.

  • Use meaningful names
  • Can't begin with a number
  • Don't use reserved keywords
  • no spaces
  • can only contain letters, numbers, $ and _. no other symbols are valid so no dashes.
  • use camelCasing.
