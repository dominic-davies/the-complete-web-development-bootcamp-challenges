Lecture 82 | Pick and Mix CSS Selectors

More CSS

Apply styles more specifically by combining selectors

Very Simple examples

selector1, selector2 {} could be h1, p {color:red}

Hierarchical selectors (spacing is crucial).

selector1 selector2 {}

1st is parent then child.

example: div h1 {color: red}

all the h1's inside a div. This is very broad. So could add (use) a id on the h1 and use a class of the parent. Actually the id might be a bit pointless here as there can be only one on a page which you could just target.

example: .container-fluid #title {color: red}

read the hierarchy right to left. so all h1's in a div (above). right is child and parent is left.

Combined Selectors

selector1.selector2 {}

selector1#selector2 {}

They have to occur in the same element. When there is no space we read left to right. h1#title, the h1 that has a id of title. You can't use these with children. Must be directly connected, div.container might be a good way of picturing this as they are always directly related at least in bootstrap.
