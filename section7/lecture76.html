You can nest items to show them on top of another item. "children sit on top of their parent element".

elements coded higher up the html document are further back while elements below are further forward. If we made boxes absolute (out of flow of document) you would have the last box blue sitting on top of yellow, which is sitting on top of red.

(bad practice) changing stacking order by moving the html elements

With the position of the colored boxes set, we can now move the blue div above the yellow div in the markup. So it's red, blue then yellow. So yellow shows on top.

(Best Practise) Z-index only works if elements are positioned like relative, absolute and fixed.

Default for all elements is z-index: 0 (static). 1 shows above other elements, -1 shows behind elements. Here I have blue 10, yellow 3, red 2.
