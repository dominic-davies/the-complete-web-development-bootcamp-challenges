Media Query Breakpoints

More CSS

Make it responsive.

You can do nothing and design a website desktop only but you will be penalised by Google etc.. especially as most people browse when on mobile. A giant company like facebook can do mobile site as well (m.facebook.com) but not feasable for most.So make responsive websites.

Can do this using css only: Media Query Breakpoints.

Syntax.

@media (type) (feature)

many types like screen, print, speech.

for screen type we are checking to see what size screen the site is being rendered in.

@media screen (min-width: 900px) then change something.

We override the existing code when screen size is 900px and higher.

viewports

You have device width (fixed), browser size which we can change. The viewport is simply the size of the screen your website is being displayed on.

max-width and min-width

min-width 900px and greater. when lower it reverts to normal.

@media screen (min-width: 900px) and (max-width: 1000px)

that example only leaves a gap of 100px. So you can target a particular device/viewport.
